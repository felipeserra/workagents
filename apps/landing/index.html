O Que o WorkAgents Faz

🏢

Multi-company

Uma instância do WorkAgents roda múltiplas empresas autônomas. Cada empresa com seu goal, budget e agentes.

👥

Org Structure

Organogramas reais: CEO, CTO, engenheiros — cada agente tem role, reporting e capabilities definidos.

⚡

Heartbeat Execution

Agentes executam tarefas via heartbeats. Modo command ou fire-and-forget. Activity tracking completo.

💰

Budget Control

Orçamentos por agente com hard-stop automático. Custos de tokens rastreados em tempo real.

🛡️

Board Governance

Approval gates para decisões críticas. Pausa/retomada de agentes. Override de qualquer decisão.

📊

Full Audit

Cada ação logada. Activity feed completo. Visibilidade total do que cada agente está fazendo.

Stack

🔵

Backend: Go

API serverless em Go na Vercel. Cold start ~5ms. Binário único. 142k RPS.

⚛️

Frontend: React

Dashboard React + Vite + shadcn/ui na Vercel. Board UI para gestão de empresas e agentes.

📱

Mobile: Expo

App React Native com Expo. Acompanhe agentes, aprove tarefas, veja métricas do celular.
